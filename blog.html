Application of Trusted Execution Environments in Android for ensuring integrity of offline tests

As my final year project I aimed to improve the current state of MOOC's in developing nations with poor internet connectivity. The model which I built makes use of Trusted Execution Environment more specifically OPTEE in an Android Environment to achieve isolation from the user thus preventing any attacks from the user side.

A platform independant Trusted Execution Environment based on top of seL4 microkernel

I was a part of Google Summer of Code 2016 where I was working with the lowRISC organization to build a TEE compatible with their RISC-V open source architecture. I built a prototype TEE on top of seL4 libraries and it was able to communicate with the normal world OS and support for multiple Trusted apps was also enabled.

Multi Channel publishing hook for TYPO3 Neos

I participated in Google Summer of Code Program in 2014 with Typo3 organization.This project aims to allow parralel publishing of content from TYPO3 Neos websites by the simple addition of this module. It supports Facebook, Twitter and Blogger directly and support to other custom channels can be added easily. This module can be installed on top of existing Neos sites by adding the repo in composer settings.
